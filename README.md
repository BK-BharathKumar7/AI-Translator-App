# AI Translator App

## Overview

The AI Translator App is a multilingual translation application developed using Python and Streamlit. It enables users to translate text between different languages in real time using machine translation techniques.

The application provides a simple and interactive interface for language translation and supports multiple global languages.

## Features

* Real-time text translation
* Multiple language support
* Character count display
* Download translated text
* User-friendly web interface
* Error handling for translation failures

## Technologies Used

* Python
* Streamlit
* Deep Translator
* Natural Language Processing (NLP)
* Machine Translation

## Supported Languages

* English
* Tamil
* Hindi
* French
* German
* Spanish
* Japanese
* Chinese (Simplified)

## Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/AI-Translator-App.git

Navigate to the project folder:

cd AI-Translator-App

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py

## Example

Input:
Hello, how are you?

Output (Tamil):
வணக்கம், நீங்கள் எப்படி இருக்கிறீர்கள்?
